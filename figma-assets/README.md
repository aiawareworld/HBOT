# Figma Assets for H-Bot About Us Page

This folder contains all Figma development files and assets for the H-Bot About Us page.

## Folder Structure

### `/images/`
Contains exported images from Figma design:
- Page screenshots
- Component exports
- Icon assets
- Background images

### `/dev-specs/`
Contains development specifications:
- Design tokens
- Spacing guidelines
- Typography specifications
- Color palette
- Component specifications

## Design Specifications

### Color Palette
- Primary Blue: `#0883fd`
- Light Blue: `#85cdfb`
- Dark Navy: `#010510`
- White: `#ffffff`

### Typography
- Font Family: Readex Pro
- Weights: 300, 400, 500, 600, 700

### Breakpoints
- Desktop: 1440px+
- Laptop: 1024px - 1440px
- Tablet: 768px - 1024px
- Mobile: 320px - 768px

### Spacing Scale
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px
- 3xl: 64px
- 4xl: 96px

## Usage
These assets are used as reference for the main implementation in the root directory files:
- `index.html` - Main page structure
- `styles.css` - Styling and design system
- `script.js` - Interactive functionality
